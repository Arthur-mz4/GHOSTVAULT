## 🎯 **Option 4: Google Safe Browsing** (Optional)

### **Why This One:**
- ✅ Google's own database
- ✅ 10,000 requests/day
- ✅ Very accurate
- ⚠️ Slightly more setup

### **How to Get API Key:**

1. **Go to:** https://console.cloud.google.com

2. **Sign in** with Google account

3. **Create new project:**
   - Click "Select a project" → "New Project"
   - Name it "GhostGuard"
   - Click "Create"

4. **Enable Safe Browsing API:**
   - Search for "Safe Browsing API"
   - Click "Enable"

5. **Create API Key:**
   - Go to "Credentials" (left menu)
   - Click "Create Credentials" → "API Key"
   - Copy the key!

6. **Done!** Paste in GhostGuard app

**Time:** 5 minutes
