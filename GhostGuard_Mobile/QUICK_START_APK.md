## 📤 HOW TO SHARE WITH SPONSORS:

### **EASIEST METHOD: Google Drive**

1. **Upload APK to Google Drive**
   - Drag APK file to Google Drive
   - Right-click → Share → "Anyone with link"
   - Copy link

2. **Create QR Code**
   - Go to: https://www.qr-code-generator.com/
   - Paste Google Drive link
   - Generate QR code
   - Download and print (A4 size)

3. **At Presentation**
   - Display QR code on screen
   - Sponsors scan with phone camera
   - They download and install APK
   - Done! ✅
